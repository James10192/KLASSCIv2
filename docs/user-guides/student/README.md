## Introduction

Ce guide explique comment utiliser le système de suivi des présences ESBTP en tant qu'étudiant. Il vous aidera à marquer votre présence efficacement et à suivre votre historique de présence.

## Accès au Système

1. Connectez-vous à l'URL : `http://votre-domaine.com/login`
2. Utilisez vos identifiants étudiant
3. En cas de problème de connexion, contactez votre délégué ou le secrétariat

## Marquage des Présences

### Procédure Standard

1. Attendez que l'enseignant communique le code de présence
2. Connectez-vous à l'application
3. Entrez le code dans le champ prévu
4. Confirmez votre présence

### Délai de Marquage

-   Le code est valide pendant une durée limitée
-   Marquez votre présence dès que possible
-   En cas de retard, contactez l'enseignant

## Mode Hors Ligne

### Utilisation Sans Connexion

1. Activez le mode hors ligne avant de perdre la connexion
2. Enregistrez votre présence localement
3. Les données se synchroniseront automatiquement au retour de la connexion

### Bonnes Pratiques

-   Gardez l'application à jour
-   Activez le mode hors ligne si nécessaire
-   Vérifiez la synchronisation des données

## Consultation des Présences

### Historique Personnel

-   Consultez vos présences par cours
-   Vérifiez vos statistiques de présence
-   Identifiez les cours manqués

### Justificatifs d'Absence

-   Soumettez vos justificatifs en ligne
-   Suivez l'état de vos demandes
-   Consultez les réponses des enseignants

## Support

### Problèmes Techniques

1. Vérifiez votre connexion internet
2. Essayez le mode hors ligne
3. Contactez le support technique si nécessaire

## Bonnes Pratiques

### Sécurité

-   Ne partagez jamais vos identifiants
-   Changez régulièrement votre mot de passe
-   Déconnectez-vous après utilisation

### Présence aux Cours

-   Arrivez à l'heure
-   Ayez votre téléphone chargé
-   Signalez tout problème rapidement

## Questions Fréquentes

1. **Que faire si j'ai oublié mon mot de passe ?**

    - Utilisez la fonction "Mot de passe oublié"
    - Contactez le secrétariat

2. **Que faire si le code ne fonctionne pas ?**

    - Vérifiez que vous avez entré le bon code
    - Demandez à l'enseignant de répéter le code
    - Utilisez le mode hors ligne si nécessaire

3. **Comment justifier une absence ?**
    - Connectez-vous à votre compte
    - Accédez à la section "Justificatifs"
    - Soumettez votre justificatif avec les documents nécessaires
