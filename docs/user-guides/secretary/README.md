## Introduction

Ce guide détaille les fonctionnalités disponibles pour les secrétaires du système de suivi des présences ESBTP. En tant que secrétaire, vous jouez un rôle crucial dans la gestion quotidienne des présences et l'assistance aux enseignants.

## Accès au Système

1. Connectez-vous à l'URL : `http://votre-domaine.com/login`
2. Utilisez vos identifiants secrétaire
3. En cas de problème de connexion, contactez l'administrateur système

## Tableau de Bord

Le tableau de bord secrétaire affiche :

-   Liste des enseignants actifs
-   Codes de présence récents
-   Notifications importantes
-   Demandes d'assistance en attente

## Gestion des Codes de Présence

### Codes Oubliés

1. Accédez à la section "Gestion des Codes"
2. Sélectionnez "Générer Code de Secours"
3. Choisissez l'enseignant concerné
4. Générez et communiquez le code

### Historique des Codes

-   Consultez l'historique des codes générés
-   Filtrez par date et enseignant
-   Exportez les données si nécessaire

## Assistance aux Enseignants

### Procédure pour Code Oublié

1. Vérifiez l'identité de l'enseignant
2. Accédez au système de génération de code
3. Générez un nouveau code
4. Communiquez le code de manière sécurisée

### Support Technique de Base

-   Assistance pour la connexion
-   Guide pour le mode hors ligne
-   Redirection vers le support technique si nécessaire

## Bonnes Pratiques

### Sécurité

-   Ne partagez jamais vos identifiants
-   Vérifiez toujours l'identité des enseignants
-   Déconnectez-vous après utilisation

### Communication

-   Utilisez les canaux officiels
-   Documentez les incidents
-   Informez les administrateurs des problèmes récurrents
