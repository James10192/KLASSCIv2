## Introduction

Ce guide détaille toutes les fonctionnalités disponibles pour les administrateurs du système de suivi des présences ESBTP. En tant qu'administrateur, vous avez accès à l'ensemble des fonctionnalités du système.

## Accès au Système

1. Connectez-vous à l'URL : `http://votre-domaine.com/login`
2. Utilisez vos identifiants administrateur
3. Activez l'authentification à deux facteurs si nécessaire

## Procédures de Secours

### Codes Oubliés

1. Accédez à la section "Codes Oubliés"
2. Sélectionnez l'enseignant concerné
3. Générez un nouveau code
4. Communiquez le code de manière sécurisée

### Problèmes de Connexion

1. Vérifiez l'état du serveur
2. Activez le mode maintenance si nécessaire
3. Utilisez les procédures de secours
