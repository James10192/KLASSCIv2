## Introduction

Ce guide détaille les fonctionnalités disponibles pour les enseignants dans le système de suivi des présences ESBTP. Il vous aidera à gérer efficacement les présences de vos étudiants, même en cas de problèmes de connexion.

## Accès au Système

1. Connectez-vous à l'URL : `http://votre-domaine.com/login`
2. Utilisez vos identifiants enseignant
3. En cas de problème, contactez le secrétariat

## Gestion des Présences

### Obtention du Code de Présence

1. Sur votre tableau de bord, cliquez sur "Générer Code"
2. Sélectionnez le cours concerné
3. Le code s'affiche à l'écran
4. Communiquez le code aux étudiants

### Suivi en Temps Réel

-   Visualisez les présences en direct
-   Identifiez les étudiants manquants
-   Gérez les retards

### Modification des Présences

-   Correction des erreurs
-   Ajout de justificatifs
-   Gestion des cas particuliers

## Mode Hors Ligne

### Activation du Mode Hors Ligne

1. Avant de perdre la connexion, activez le mode hors ligne
2. Les données seront stockées localement
3. Synchronisation automatique au retour de la connexion

### Utilisation Hors Ligne

-   Génération de codes hors ligne
-   Enregistrement local des présences
-   Résolution des conflits à la synchronisation

## Procédures de Secours

### Code Oublié ou Perdu

1. Contactez le secrétariat
2. Fournissez vos informations d'identification
3. Recevez un nouveau code
4. Notez le nouveau code en lieu sûr

### Problèmes Techniques

1. Vérifiez votre connexion internet
2. Utilisez le mode hors ligne si nécessaire
3. Contactez le support technique si le problème persiste

## Bonnes Pratiques

### Sécurité

-   Changez régulièrement votre mot de passe
-   Ne partagez jamais vos identifiants
-   Déconnectez-vous après chaque session

### Gestion des Présences

-   Générez le code au début du cours
-   Vérifiez les présences pendant le cours
-   Signalez les anomalies rapidement
