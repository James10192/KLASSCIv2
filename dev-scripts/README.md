# Dossier dev-scripts

Ce dossier contient tous les fichiers de développement, test et scripts temporaires qui ont été déplacés de la racine du projet pour une meilleure organisation.

## Structure

- `analysis-scripts/` - Scripts d'analyse des données et extraction (Python, JSON)
- `backup-files/` - Anciens fichiers de sauvegarde et versions précédentes
- `backup-scripts/` - Scripts de sauvegarde automatisés
- `documentation/` - Documentation technique et guides de développement
- `memory-bank/` - Archives de contexte et briefs de projet
- `migration-scripts/` - Scripts de migration de données
- `project-memory/` - Historique des corrections et rapports de développement
- `python-scripts/` - Scripts Python pour l'analyse et l'extraction de données
- `session-notes/` - Notes de sessions de développement (fichiers TXT)
- `temp-files/` - Fichiers temporaires et de test
- `test-files/` - Fichiers de test HTML, PHP et autres

## Utilisation

Ces fichiers ont été organisés pour garder la racine du projet propre tout en préservant l'historique de développement et les outils utiles pour le débogage et la maintenance.

## Attention

Ne pas supprimer ce dossier car il contient des informations importantes sur l'historique du projet et des outils de diagnostic qui peuvent être utiles pour la maintenance future.
