# Task ID: 21
# Title: Implement Department and Laboratory View Models, Controllers, and Sidebar Navigation
# Status: pending
# Dependencies: None
# Priority: medium
# Description: Create view models and controllers for Department and Laboratory modules, and integrate them into the application's sidebar navigation in app.blade.php.
# Details:
This task involves several components:

1. Create view models for both Department and Laboratory:
   - Implement DepartmentViewModel and LaboratoryViewModel classes
   - Include necessary properties for data binding
   - Implement validation rules and error handling
   - Add any required data transformation methods

2. Develop controllers for both modules:
   - Create DepartmentController and LaboratoryController classes
   - Implement CRUD operations (index, create, store, show, edit, update, destroy)
   - Add proper request validation
   - Implement appropriate response handling and redirects
   - Ensure proper authorization checks are in place

3. Update the sidebar navigation in app.blade.php:
   - Add new menu items for Department and Laboratory
   - Use appropriate icons that match the application's design system
   - Implement proper routing links to the new controllers
   - Ensure the sidebar items are only visible to users with appropriate permissions
   - Maintain consistent styling with existing sidebar elements

4. Create necessary blade views for both modules:
   - index.blade.php (listing view)
   - create.blade.php and edit.blade.php (forms)
   - show.blade.php (details view)

Ensure all components follow the existing application architecture patterns and coding standards.
